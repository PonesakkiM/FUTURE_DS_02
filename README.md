SOCIAL MEDIA CAMPAIGN PERFORMANCE Dashboard -Power BI
 I analyzed Facebook & Instagram ad campaign data and built a complete Power BI Marketing Performance Dashboard.
 Project Overview
The goal of this project is to evaluate the performance of various social media ad campaigns using key marketing metrics such as:
Spend
Impressions
Clicks
Leads
Orders
Revenue
CPL (Cost per Lead)
CPA (Cost per Acquisition)
ROAS
The dashboard helps measure which campaigns performance
 Objectives

1. Build an interactive Power BI Dashboard
2.Analyze Facebook & Instagram campaign performance
3.Track spend efficiency: CPL, CPA
4. Visualize campaign funnel
5.Compare categories: Social, Search, Media, Influencer
6.Use filters for campaign, category & spend range
 Dataset Details
The dataset includes multiple marketing campaign records with the following columns:

date
campaign_name
category (social / influencer / search / media)
impressions
spent
clicks
leads
orders
revenue
CPL
CPA
ROAS

Tools Used
Power BI	Dashboard creation
Excel / CSV	Data cleaning
Power Query	Transformations
DAX	KPI calculations

 Dashboard Features

KPI Cards: Spend, Impressions, Revenue, CPL, CPA

Gauge Chart: Revenue Target progress

Donut Chart: CPL / CPA visual

Funnel Chart: Impressions → Clicks → Leads → Orders

Bar Chart: Orders by category

Line Chart: CPC/CTR trend

Filters:
Campaign
Category
Spend Range Slider

Platform Icons: Facebook, Instagram, YouTube
